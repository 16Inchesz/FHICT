COMMUNICATION PATTERNS
Technical Report
======================

Introduction
------------
In this folder you find a technical report of the design of two communication
patterns, namely sender and receiver, that is useful for designing communication
protocols for reliable and robust sytems.

Please, read the technical report. The report contains valuable content. The
designs were guided by states and state-transitions.

How to Install
--------------
In Visual Studio Code, add folder "docs" to your explorer with "Add Folder to 
Workspace..." (right mouse button in workspace area) and select the 
"Technical report.adoc" file. Open the preview tab (using F1). The state diagram
is shown. If it is not shown then you need to enable the Use_kroki setting in
the AsciiDoc extension.
